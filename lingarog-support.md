# Lingarog Support

Lingarog is a global phrasebook: pick a source language and a target language, browse ready-made everyday sentences across categories, hear pronunciation, and copy either side. Every sentence exists in every supported language.

## Using the app

- Tap the language icon (top right) to choose what you speak and what you want to say.
- Browse categories, or use search to find a sentence.
- Tap a sentence to see both languages, hear the target pronounced, and copy either side.

## Privacy

Lingarog does not collect personal data in version 1.0. See [Lingarog Privacy Policy](lingarog-en.md).
