# Informativa sulla privacy di Live Shader

*Data di efficacia: 3 maggio 2026*

Live Shader non raccoglie, conserva, vende o condivide dati personali.

La versione 1.0 non raccoglie informazioni personali, analisi d'uso, diagnostica, identificatori pubblicitari, dati di contatto, posizione, foto, audio o contenuti utente.

Live Shader non traccia gli utenti su app o siti web di altre aziende. Non richiede un account e non invia attivita dell'app a un server.

Supporto: [Live Shader Support](live-shader-support.md).
