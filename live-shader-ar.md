# سياسة خصوصية Live Shader

*تاريخ السريان: 3 مايو 2026*

لا يجمع Live Shader البيانات الشخصية ولا يخزنها ولا يبيعها ولا يشاركها.

لا يجمع الإصدار 1.0 معلومات شخصية أو تحليلات استخدام أو بيانات تشخيصية أو معرفات إعلانية أو معلومات اتصال أو بيانات موقع أو صورًا أو صوتًا أو محتوى مستخدم.

لا يتتبع Live Shader المستخدمين عبر التطبيقات أو المواقع المملوكة لشركات أخرى. لا يتطلب التطبيق حسابًا ولا يرسل نشاط التطبيق إلى خادم.

للدعم: [Live Shader Support](live-shader-support.md).
