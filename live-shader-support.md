# Live Shader Support

Live Shader is a native Apple TV visual app for realtime Metal shader effects.

## Controls

- Use the Siri Remote directional controls to move through shader categories.
- Select a card to switch effects.
- Select the active card again to toggle fullscreen clean mode.
- Press Play/Pause to toggle auto-cycle and enter clean mode.
- Press Back/Menu to show or hide controls.

## Privacy

Live Shader does not collect personal data in version 1.0.

## Contact

For App Store support, contact the developer through the App Store listing or open an issue on the relevant GitHub project when available.
